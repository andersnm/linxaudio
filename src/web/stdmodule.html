Value modules - Control other modules

adsrvalue Generates values for an ADSR envelope in the range 0..1. Values are emitted every 16 samples
arithmeticvalue Performs arithmetic operations on two values to produce a single value. Add, subtract, multiply, divide, modulus
envscale Scale a value from a range to another
lfovalue Generate low frequency values
midinotesplit Generate trigger, velocity and note values from midi note input
note frequency Convert note value to frequency
uservalue_* Various simple value emitters for controlling several pins with a single (user controlled) pin

Audio modules - Basic, Mono

biquad_filter Biquad filter
biquad_eq Biquad equaliser
biquad_trifilter Filters from FSM infector
clip Audio clipping w/pre and post amp
delay Requires subgraph
distortion Audio distortion
gain Audio gain
oscbuffer Oscillator. Generates sine, square, saw, inv saw, triangle, gaussian noise audio buffers
polycontainer Requires subgraph. Adds polyphonic support to the subgraph contents. Clones the subgraph for each voice up to 8 simultaneous voices
svf State variable filter
